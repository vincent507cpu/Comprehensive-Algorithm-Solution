# Description
Implement function `ToLowerCase()` that has a string parameter str.And convert the uppercase letters in the string to lowercase letters, and then return the new string.

# Example
## Example 1:
```
Input: "Hello"
Output: "hello"
```
## Example 2:
```
Input: "here"
Output: "here"
```
## Example 3:
```
Input: "LOVELY"
Output: "lovely"
```
