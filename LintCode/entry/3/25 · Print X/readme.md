# Description
Given a positive integer N, you need to return a list string like this.

# Example
## Example 1:
```
Input：1
Output：
[
"X"
]
```
## Example 2:
```
Input：2
Output：
[
"XX",
"XX"
]
```
## Example 3:
```
Input：3
Output：
[
"X X",
" X ",
"X X"
]
```
## Example 4:
```
Input：4
Output：
[
"X  X",
" XX ",
" XX ",
"X  X"
]
```
## Example 5:
```
Input：5
Output：
[
"X   X",
" X X ",
"  X  ",
" X X ",
"X   X"
]
```
