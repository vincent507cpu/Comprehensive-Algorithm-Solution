Merge K sorted interval lists into one sorted interval list. You need to merge overlapping intervals too.

# Example
## Example1
```
Input: [
  [(1,3),(4,7),(6,8)],
  [(1,2),(9,10)]
]
Output: [(1,3),(4,8),(9,10)]
```
Example2
``
Input: [
  [(1,2),(5,6)],
  [(3,4),(7,8)]
]
Output: [(1,2),(3,4),(5,6),(7,8)]
```
