Implement a stack by two queues. The queue is first in first out (FIFO). That means you can not directly pop the last element in a queue.

# Example
## Example 1:
```
Input:
push(1)
pop()
push(2)
isEmpty() // return false
top() // return 2
pop()
isEmpty() // return true
```
