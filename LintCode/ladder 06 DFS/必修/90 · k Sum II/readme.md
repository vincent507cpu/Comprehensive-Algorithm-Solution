# Description
Given `n` unique postive integers, number `k` (`1<=k<=n`) and target.

Find all possible `k` integers where their sum is target.

# Example
## Example 1:

Input:
```
array = [1,2,3,4]
k = 2
target = 5
```
Output:
```
[[1,4],[2,3]]
```
Explanation:

1+4=5,2+3=5

## Example 2:

Input:
```
array = [1,3,4,6]
k = 3
target = 8
```
Output:
```
[[1,3,4]]
```
Explanation:

1+3+4=8
